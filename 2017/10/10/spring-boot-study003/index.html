SpringBoot使用模板来构建web服务

SpringBoot中进行web服务的构建,最好采用thymeleaf来进行开发,Spring 官网也推荐使用thymeleaf框架来进行开发。使用thymeleaf开发需要先引用依赖

<dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-thymeleaf</artifactId>
    </dependency>

依赖添加完毕以后,下面开始编写一个网页模板；

<html xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="utf-8" />
</head>
<body>
    <a>测试例子</a>
</body>
</html>

这样直接调用是可以了,但是现在想向页面中传递数据如何处理,具体的见如下的例子:

在Controller中增加如下代码:

@RequestMapping(value = "/test/param", method = RequestMethod.GET)
public String test2(Model model){
    model.addAttribute("content","测试内容");
    return "demo1";
}

@RequestMapping(value = "/test/param/1", method = RequestMethod.GET)
public ModelAndView test3(){
    ModelAndView mv = new ModelAndView("demo1");
    mv.addObject("content","测试内容");
    return mv;
}

增加里如下代码后,在templates中增加如下页面:

<html xmlns:th="http://www.thymeleaf.org">
<head>
    <meta charset="utf-8" />
</head>
<body>
    <a th:text="${content}"></a>
</body>
</html>
