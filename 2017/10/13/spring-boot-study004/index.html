Spring Boot操作数据库

SpringBoot整合jdbc

spring整合jdbc,让开发更加的简单方便,使用起来也很方便,配置起来也比较简单,首先需要增加相应的依赖:

<dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-jdbc</artifactId>
    </dependency>
    <dependency>
        <groupId>mysql</groupId>
        <artifactId>mysql-connector-java</artifactId>
        <version>5.1.44</version>
    </dependency>

这两个依赖添加以后,还需要在增加一个依赖,因为spring-boot中只是做了一个数据库操作的基本框架,但是数据库的连接池的管理是没有的,需要在使用时选择,这里选择了阿里的druid来处理的。

<dependency>
        <groupId>com.alibaba</groupId>
        <artifactId>druid</artifactId>
        <version>1.0.19</version>
    </dependency>

依赖添加好以后,剩下的是需要增加配置内容,下面为简单的配置内容为:

spring.datasource.url=jdbc:mysql://localhost:3306/spider
spring.datasource.username=root
spring.datasource.password=root
spring.datasource.driver-class-name=com.mysql.jdbc.Driver

依赖添加好以后,剩下的就是在注册数据库连接池的bean,如下:

@Autowired
private Environment env;

@Bean
public DataSource dataSource() {
    DruidDataSource dataSource = new DruidDataSource();
    dataSource.setUrl(env.getProperty("spring.datasource.url"));
    dataSource.setUsername(env.getProperty("spring.datasource.username"));//用户名
    dataSource.setPassword(env.getProperty("spring.datasource.password"));//密码
    dataSource.setDriverClassName(env.getProperty("spring.datasource.driver-class-name"));
    dataSource.setInitialSize(2);
    dataSource.setMaxActive(20);
    dataSource.setMinIdle(0);
    dataSource.setMaxWait(60000);
    dataSource.setValidationQuery("SELECT 1");
    dataSource.setTestOnBorrow(false);
    dataSource.setTestWhileIdle(true);
    dataSource.setPoolPreparedStatements(false);
    return dataSource;
}

SpringBoot 整合mybatis

mybatis和springboot的整合现在很简单,可以使用注解来操作或则是采用简单的xml配置方式来处理。

整合现在也很简单,只需要增加一个依赖就可以了,具体的依赖如下:

<dependency>
        <groupId>org.mybatis.spring.boot</groupId>
        <artifactId>mybatis-spring-boot-starter</artifactId>
        <version>1.3.0</version>
    </dependency>

首先需要制定mapper的指定的扫描地址,配置内容如下：

@MapperScan("com.spring.study.web.mybatis")
@SpringBootApplication
public class SpringBootWebApplication {}

其中使用注解来操作,主要的注解介绍如下:

@Select 是查询类的注解，所有的查询均使用这个
@Result 修饰返回的结果集，关联实体类属性和数据库字段一一对应，如果实体类属性和数据库属性名保持一致，就不需要这个属性来修饰。
@Insert 插入数据库使用，直接传入实体类会自动解析属性到对应的值
@Update 负责修改，也可以直接传入对象
@delete 负责删除

采用xml配置来操作

用xml配置,首先需要指定mybatis的配置文件的地址,需要在application.properties中增加如下配置:

mybatis.config-locations=classpath:mybatis/mybatis.xml
mybatis.mapper-locations=classpath:mybatis/mapper/*.xml

配置好这些以后,需要指定对应的mapper.xml:

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE mapper PUBLIC "-//mybatis.org//DTD Mapper 3.0//EN" "http://mybatis.org/dtd/mybatis-3-mapper.dtd">
<mapper namespace="com.spring.study.web.mybatis.UserMapper2">
<resultMap id="BaseResultMap" type="java.lang.String">
    <result column="name" property="name" jdbcType="VARCHAR"/>
</resultMap>

<select id="select" resultMap="BaseResultMap">
  select name from user where name = #{name}
</select>
<select id="selectAll" resultMap="BaseResultMap">
    select name
    from user
</select>
</mapper>
