下面开始查看完集合中添加一个元素的动作ConcurrentHashMap做了哪些动作？

通过给定需要插入到map中的k和v,通过k进行hash加密下,通过这个hash值来获取到相应的在Segment[]数组中的对应的下标,如果不存在就在数组Segment[]对应下标位置上的创建一个新的Segment。

之后再向这个Segment中写入数据,具体的写入操作很简单就是获取到Segment后,可以找到Segment包装的HashEntry集合,在对这个集合进行加锁,实现并发情况下的数据写入。
